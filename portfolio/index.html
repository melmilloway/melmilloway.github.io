I work with a variety of tools to create learning experiences. Tools I work with include: Adobe Creative Suite, Camtasia, Sketch, front-end frameworks like Bootstrap, the Adapt Authoring Tool/Framework, and Articulate Storyline. Check out some of my work below.

  • Educational Coffee Cards

    A card pattern built in Storyline 360. Icons from Freepik and copy in cards from Coffee Crossroads.

  • Is It Edible?

    A practice activity built in Storyline 360. Players must identify which plants are edible versus non-edible. Best viewed on desktop.

  • Is It Edible? UI Design

    The mockup for a practice activity that was then built in Storyline 360. Mockup created with Adobe XD.

  • Presto Pasta Rewards Program Guide

    The Rewards Program Guide site was built from a template in the simple static site generator, Jekyll. The site is used for planning xAPI statements in a scenario based workshop on xAPI.

  • Presto Pasta UI Design

    The Presto Pasta Mockups show screens for a desktop and mobile rewards program guide and a mock POS (Point of Sale) system. After I create the designs I use them as a guide to develop the interfaces. Mockup created with Adobe XD.

  • Scrum Master Trivia

    An xAPI enabled Storyline 360 project.

  • Scrum Diaries Video

    A an animated explained video created in After Effects for the Scrum Master Trivia project.

  • Scrum Master Trivia UI Design

    A mockup for a Storyline 360 module, created in Adobe XD.

  • Scenes Around Seattle

    A site with VR tours of Seattle. The 360 VR environments are xAPI enabled and track user's interactions to a database.

  • Scenes Around Seattle UI Design

    A mockup of my VR and xAPI enabled project, Scenes Around Seattle. Mockup created with Sketch.

  • Rad Quiz

    A responsive quiz template created in the Adapt Framework.

  • Espresso Matchmaker

    An HTML5 Storyline example that reports your choices to an LRS using xAPI.

  • A Guide to Safe Food Handling

    A responsive landing page example that was created in the Adapt Authoring Tool.

  • The Unconscious Bias Game

    A mini game prototype created with Twine and Javascript. This project reports your choices to an LRS using xAPI.
